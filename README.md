### 二、作品包含
源码+数据库+全套环境和工具资源+本地部署教程

### 三、项目技术

数据库管理工具：Navicat10+

Python：Python3.7

前端Nodejs：14

浏览器：谷歌浏览器

### 五、项目介绍
项目编号：python058

音乐平台系统用于整合音乐资源，为用户提供音乐播放、搜索、推荐、社交互动及创作者管理等一站式服务。
该音乐平台系统具备前台用户浏览（音乐、海报、歌手信息浏览，论坛互动 ）与后台管理（音乐、分类、海报、歌手数据增删改查及统计 ）功能，支持用户与管理员不同操作场景。
